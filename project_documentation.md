# Competitor Intelligence Dashboard Documentation

This project is a real-time competitor intelligence tracking system. It automatically scrapes target competitors' websites, pricing pages, careers boards, and blogs, analyzes the changes using Google Gemini with structured outputs, sends automated alerts to WhatsApp via Twilio, and renders everything in a beautiful Next.js dashboard.

1. **Frontend / Backend**: Next.js 16 (App Router), React 19, TypeScript

3. **Database & ORM**: PostgreSQL, Prisma Client
4. **Proxy Scraper**: Bright Data Web Scraper API (for reliable scraping without bot-detection issues)
5. **Generative AI**: Google Gemini API (`gemini-2.5-flash`, `gemini-2.5-pro`, `gemini-3.5-flash` with failover mechanisms)
6. **Notifications**: Twilio WhatsApp API
7. **Scheduling**: Node-Cron (for automated periodic background checks)

## рџ—„пёЏ Database Schema (`schema.prisma`)

The database consists of 4 main models:

*   **Competitor**: Stores target competitor companies with names, website domains, and targeted page links (pricing, careers, blog).
*   **Change**: Tracks individual delta detections (e.g. pricing edits, feature additions, new job postings). Includes categories (`pricing`, `feature`, `blog`, `jobs`, `other`) and severity tiers (`critical`, `high`, `medium`, `low`).
*   **Insight**: Stores cross-competitor market intelligence reports (trends, alerts, opportunities) generated by Gemini.
*   **ScrapeRun**: Audit log history of scanner execution runs, tracking durations, status results, and metrics of discovered updates.

## рџ¤– AI & Integration Workflows

### 1. Crawler Engine (`lib/scanner.ts`)
*   Triggered via POST to `/api/scan`.
*   Iterates through all competitors.
*   Fetches designated URLs via the **Bright Data** scraper proxy. If blocked or empty, utilizes predefined mock templates to ensure stable demo presentation.
*   Performs content comparison/deduplication against the last recorded change to prevent double-logging.
*   Passes new content to Gemini for classification.

### 2. Gemini Processor (`lib/gemini.ts`)
*   Ensures maximum system uptime using an automated failover model chain: `gemini-2.5-flash` вћЎпёЏ `gemini-2.5-pro` вћЎпёЏ `gemini-3.5-flash`.
*   Applies structured response JSON Schemas to enforce type safety on classification and strategic summaries.

### 3. Twilio WhatsApp Dispatch (`lib/twilio.ts`)
*   Formats discovered changes and strategic AI recommendations into a markdown list message.
*   Dispatches instantly to WhatsApp, attaching target competitor logos when available.

Create a `.env.local` file with the following variables:
```env

BRIGHT_DATA_API_KEY="your-bright-data-api-key"
```

### 2. Run Database Migrations & Seed
```bash

```

```bash

```
